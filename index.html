Who started computer games?

A student at the University of Utah, who's name was Nolan Bushnell discovered computer games in the mid-1960s. The first computer game was developed at MIT in 1962, A game called Spacewar! which originally ran on a PDP-1 computer the size of a family car. These days computer games run on more compact computers.

Fun Fact

The computer game Spacewar! was used to test the function of computers coming off the computer assembly line
